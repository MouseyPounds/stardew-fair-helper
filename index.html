Stardew Fair Helper

About

This app helps to prepare for the Grange Display at the Fall Festival in Stardew Valley. It will calculate the score for manually-entered items and also search through a save file to recommend which item combination would give the highest possible score. Please report any bugs, suggestions, or other feedback to the topic in the official forums. (TODO: Fix URL to actual topic once it is created)

The app is written in Javascript and uses jQuery and Select2; it is hosted on GitHub Pages at https://mouseypounds.github.io/stardew-fair-helper/ and the source code repository is https://github.com/MouseyPounds/stardew-fair-helper.

Save File

If you would like the app to scan a save file in order to recommend what items to use for the display, you can select the save file below:

Please use the full save file named with your farmer's name and a 9-digit ID number (e.g. Fred_148093307); do not use the SaveGameInfo file as it does not contain all the necessary information.

Default save file locations are:

  • Windows: %AppData%\StardewValley\Saves\
  • Mac OSX & Linux: ~/.config/StardewValley/Saves/

Item Input

This section will be filled in automatically when loading a save. It can also be edited manually to experiment with other options or to use the app without loading a save. When analyzing a multiplayer save, it is assumed that all players are online and so all players' professions are included. Note that you cannot select the Lucky Purple Shorts as a choice and the app will not suggest it either since it has a fixed special result.
